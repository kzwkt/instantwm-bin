|-----------|-----------------|
| `r`       | termfilemanager |
| `e`       | editor          |
| `n`       | filemanager     |
| `Space`   | appmenu         |
| `f`       | browser         |
| `a`       | .instantassist  |
| `F1`      | .instanthotkeys |
| `s`       | terminal        |
| `+`       | ~volup          |
| `-`       | ~voldown        |
| `Tab`     | ~caretswitch    |
| `c`       | code            |
| `y`       |instantmenu_smartrun|
| `v`       | .quickmenu      |
| h,l      | view left, right      |
| k,j      | shift view +1, -1      |
