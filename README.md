# Keybindings
root window
| Key       | Action                  |
|-----------|------------------------|
| `r`       | Launch Ranger           |
| `e`       | Open Editor             |
| `n`       | Open Nautilus           |
| `Space`   | Launch Panther          |
| `f`       | Open Firefox            |
| `a`       | Launch Instant Assist   |
| `F1`      | Open Help               |
| `s`  | Open Terminal           |
| `+`       | Increase Volume         |
| `-`       | Decrease Volume         |
| `Tab`     | Switch Caret Mode       |
| `c`       | Open Code Editor        |
| `y`       | Run Smart Command       |
| `v`       | Open Quick Menu         |
